
Predix Machine Startup Scripts
=====================================================================

-- predixmachine (Linux, Macintosh) / predixmachine.bat (Windows) --

 This file is the main startup script. To perform a 
 clean start add the argument "clean" to the executable command.
 For example:
 
 	"predixmachine.bat clean" or "predixmachine clean"
 
   
-- setvars (Linux, Macintosh) / setvars.bat (Windows) --
 This file is for setting specific variables for the predix machine configuration.
 
 -- predix.home.prs (generated) --
 This file is generated by the startup script setting the full path in a system property file for permission.
